# Abyss DiveDown
Simple lua script i made for the Roblox game DiveDown, you can use the helper functions on other games tho.

# Usage Information
When using if you want to get HumanoidRootPart just do 
```
local RootPart = GetRoot();
```
For drawing highlight on objects then to, esp.draw_highlight(object_children, color)
You should be able to understand how to use the esp.draw_name(obj or plr, lp.Name) function just fine.
